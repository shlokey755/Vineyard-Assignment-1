# Assignment 1 - Portfolio Website

## 👤 Author

**Shlok**

## 📌 Description

This project is a simple personal portfolio website created using **HTML** and **CSS**.
It was developed as part of **Assignment 1**.

The website includes basic sections to showcase personal information, hobbies, and projects.

## 🛠️ Technologies Used

* HTML
* CSS

## 📂 Files Included

* `index.html` → Main structure of the website
* `style.css` → Styling and layout of the website

## 🌐 Features

* Clean and simple design
* Navigation bar for easy access
* Sections: Home, About, Projects, Contact
* Responsive layout for different screen sizes

## 🚀 How to Run

1. Download or clone the project
2. Open `index.html` in any web browser

## 📄 Notes

This is a beginner-level project created for learning purposes.
